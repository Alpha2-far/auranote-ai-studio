# TASK-007 : Interface de Lecture & Rédaction Épurée (Zen Reader & Editor UI)

---

## 📌 1. Le Problème
L'utilisateur recherche un espace de travail visuel ultra-épuré et serein pour consulter, relire et retravailler ses réflexions sans être distrait par des éléments d'interface surchargés.

---

## 🎯 2. L'Objectif
Concevoir l'interface utilisateur web (HTML5, Vanilla CSS, Vanilla JS) intégrant la charte visuelle officielle AuraNote (Mode Obsidienne Nuit `#0D0F12`, Accents Dorés `#E2B872`, Typographie épurée) et intégrant le composant de lecture et d'édition de notes.

---

## 🏁 3. Le Résultat Attendu
Une interface Web responsive (Desktop & Mobile) centrée sur le texte avec affichage du logo officiel SVG (`assets/icon.svg`), une barre de capture "Smart Paste" visible et un lecteur/éditeur Markdown épuré.

---

## 🔍 4. Le Périmètre
- Rendu du logo SVG officiel (Monogramme A & Halo d'Aura).
- Thème CSS "Obsidienne Nuit" & Design System Auras (`css/main.css`, `css/auras.css`, `css/zen-editor.css`).
- Composants UI (`ZenReaderEditorComponent.js`, `HeaderComponent.js`, `SmartPasteButtonComponent.js`).
- Intégration d'un parseur Markdown léger pour l'affichage épuré.

---

## 🚫 5. Le Hors Périmètre
- La recherche avancée et l'export PDF (traité dans `TASK-008`).
