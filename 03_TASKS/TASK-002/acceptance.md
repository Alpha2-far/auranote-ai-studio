# TASK-002 : Critères d'Acceptation

---

## 📋 Checklist de Validation

- [x] L'utilisateur peut sélectionner ou créer le fichier `Monidée.md` via le sélecteur de fichier natif.
- [x] Le handle du fichier est correctement sauvegardé pour réutilisation durant la session.
- [x] L'ajout d'une nouvelle note concatène proprement un bloc Markdown structuré en fin de fichier.
- [x] Le bloc Markdown inclut les métadonnées (ID, Titre, Aura, Date).
- [x] En cas de refus de permission ou de navigateur incompatible, l'application informe l'utilisateur et active le fallback de sauvegarde.
