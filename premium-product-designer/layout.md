# RULE 2

Name

Single Focal Point

Classification

SYNTHESIS

Confidence

95%

Reason

The guide discusses focal points and visual hierarchy; limiting each screen to one dominant focal area is a design synthesis rather than a direct quote. :contentReference[oaicite:2]{index=2}

Rule

Every screen should have one dominant point of attention.

Everything else supports it.

Never create multiple competing heroes.

# REVIEW QUESTIONS

Before validating any layout

Can the user identify the focal point within two seconds?

Does every element have a structural purpose?

Is the reading path obvious?

Does whitespace improve understanding?

Can one element be removed without harming comprehension?

If yes,

remove it.

---

# REJECTION CONDITIONS

Reject the layout if

Two CTAs compete.

Alignment breaks without reason.

Spacing changes arbitrarily.

Hierarchy is unclear.

The eye becomes lost.

Whitespace disappears.

The interface feels crowded.

---

# FINAL PRINCIPLE

The best layout is not the one that contains the most.

The best layout is the one that requires the least effort to understand.
