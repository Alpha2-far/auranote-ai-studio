# Accent Usage

Gold exists

to reward attention.

Never fill screens with gold.

Maximum

One accent focus

per viewport.

---

# Background Rule

Background should disappear.

Never compete with content.

Dark backgrounds should feel

soft

not black.

---

# Reading Rule

Text always wins.

If a background reduces readability,

the background loses.

Never the text.

# Review Questions

Can users understand states

without reading?

Does every color have meaning?

Can one color be removed?

Is contrast sufficient?

Is the accent memorable?

Does the interface feel calm?

---

# Final Principle

Users should never think

"This interface has beautiful colors."

They should think

"This interface is effortless to understand."

That is successful color design.
