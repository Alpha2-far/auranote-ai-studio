# Typography Philosophy

Typography is the interface.

Most interfaces are made of text.

Improving typography improves almost every screen.

Never treat typography as decoration.

Typography is information architecture.

---

# Primary Goal

Typography must achieve

Understanding

before

Expression.

Beautiful typography that reduces readability

is bad typography.

---

# Reading First

Users should never notice typography.

They should notice

their understanding.

If typography becomes the center of attention,

it has failed.

# Paragraph Spacing

Paragraphs communicate ideas.

Spacing between paragraphs should be greater than spacing between lines.

Ideas deserve visual separation.

# Capital Letters

Avoid full uppercase paragraphs.

Uppercase is acceptable for

Badges

Overlines

Small labels

Navigation groups

Never for reading content.

---

# Numbers

Use tabular figures

where comparison matters.

Financial data

Tables

Statistics

Timers

Never mix proportional and tabular numbers on the same data table.

---

# Links

Links should be identifiable

without relying solely on color.

Interaction must remain obvious.

---

# Quotes

Quotes deserve visual breathing room.

Increase surrounding whitespace.

Reduce visual competition.

---

# Lists

Bullets

when order is irrelevant.

Numbers

when sequence matters.

Never create long uninterrupted text blocks.

---

# Error Messages

Be

Specific

Actionable

Calm

Never blame the user.

Bad

Invalid input.

Good

The email address is missing an "@". Please check it and try again.

---

# Empty States

Typography should explain

What happened.

Why it happened.

What happens next.

Never leave the user without direction.

---

# Writing Tone

AuraNote speaks

Clearly.

Calmly.

Precisely.

Professionally.

Never

Sarcastically.

Aggressively.

Emotionally manipulative.

Overly technical without need.

---

# Typography Review

Before approving

Can this page be read for ten minutes comfortably?

Is hierarchy obvious?

Can headings be scanned in seconds?

Is body text effortless?

Can users distinguish sections instantly?

Does typography reduce thinking effort?

If one answer is NO,

redesign.

---

# Rejection Conditions

Reject if

More than three font sizes compete.

Random font weights appear.

Centered long-form paragraphs exist.

Reading requires zoom.

Paragraphs become visual walls.

Typography competes with illustrations.

Spacing between lines is inconsistent.

---

# Final Principle

Users do not remember fonts.

They remember

whether reading felt effortless.

Typography succeeds

when understanding becomes invisible.
