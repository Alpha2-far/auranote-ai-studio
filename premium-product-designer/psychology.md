# REVIEW QUESTIONS

Before approving a design ask

What emotion appears first?

What message does the typography send?

What message do the colors send?

What message do the shapes send?

Does the interface create trust?

Does it reduce cognitive effort?

Can the design be understood before reading?

---

# REJECTION CONDITIONS

Reject if

Color has no semantic purpose.

Typography contradicts product identity.

Shapes create unintended emotions.

Decorative elements compete with content.

Animations attract more attention than information.

The interface creates anxiety.

The interface creates urgency without reason.

The interface feels visually loud.

---

# FINAL PRINCIPLE

People rarely remember every interface.

They always remember

how the interface made them feel.

Design the feeling first.

The pixels follow.
