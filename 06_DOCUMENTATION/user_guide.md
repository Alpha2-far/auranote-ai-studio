# 06_DOCUMENTATION — Guide Utilisateur AuraNote AI Studio

## 📖 Bienvenue dans AuraNote AI Studio

AuraNote AI Studio est votre carnet de réflexions intelligent ("second cerveau") conçu pour centraliser et structurer vos échanges stratégiques avec les IA (Gemini, Claude, ChatGPT).

---

## 🚀 4 Façons de Capturer une Idée

1. **Le Coller Intelligent (PC & Mobile) :**
   - Copiez la réponse de l'IA.
   - Cliquez sur le bouton "Smart Paste" dans AuraNote.
   - Votre note est automatiquement nettoyée, titrée et rangée dans son Aura.
2. **Le Partage Mobile (Smartphone) :**
   - Dans Gemini / Claude / ChatGPT, appuyez sur **Partager**.
   - Choisissez **AuraNote** dans la liste des applications.
3. **Le Bouton Navigateur (Extension PC) :**
   - Faites un clic droit sur la réponse affichée à l'écran et sélectionnez *Envoyer vers AuraNote*.
4. **L'Ordre à l'IA :**
   - Dites à votre assistant : *"Résume nos choix et envoie-les dans mon carnet AuraNote"*.
